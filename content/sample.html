BasicProperties

build.propertiesの基本設定(BasicProperties)について記述します。
(自動生成前にこれだけは押さえておきたい！)

for All

torque.project
Project名です。必須

(例)torque.project = ldb

さらに、以下の値をこのProject名と同じ値にする必要があります。

  • build-XXX.propertiesのXXXの部分
  • _project.bat内の定数「MY_PROJECT_NAME」の値

torque.database
接続するDatabaseの種類です。必須

(例)torque.database = derby

Comment上に記載されているDatabase名称から選んでコピーして設定して下さい。

  • 「mssql」は、SQLServerのことです(2000/2005区別なし)。
  • 足りてない場合は相談して下さい。

torque.targetLanguage
自動生成するClassの言語です。必須

(例)torque.targetLanguage = java

Comment上に記載されているLanguageから選んでコピーして設定して下さい。

for OM task

torque.java.dir
自動生成するSourceの出力Directoryです。

(例)torque.java.dir = ../src/main/java
