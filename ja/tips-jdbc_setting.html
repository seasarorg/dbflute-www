JDBCの設定について記述します。 


Apache Derby (derby)

'derby.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties - for JDBC task}
torque.database.driver      = org.apache.derby.jdbc.EmbeddedDriver
torque.database.url         = jdbc:derby:/pj/ws/dbflute_ldb/src/test/resources/data-derby
torque.database.schema      = LDB
torque.database.user        = ldb_user
torque.database.password    = ldb_password

※schemaは、大文字で指定しなければならないことにに注意して下さい。

※上記は、実行時にServiceを自動起動する場合の指定です。



Oracle (oracle)

'ojdbc14.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties - for JDBC task}
torque.database.driver      = oracle.jdbc.driver.OracleDriver
torque.database.url         = jdbc:oracle:thin:@localhost:1521:orcl
torque.database.schema      = LDB
torque.database.user        = ldb_user
torque.database.password    = ldb_password

※schemaは、大文字で指定しなければならないことにに注意して下さい。



SQLServer (mssql)

'sqljdbc.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties - for JDBC task}
torque.database.driver      = com.microsoft.sqlserver.jdbc.SQLServerDriver
torque.database.url         = jdbc:sqlserver://localhost:1433;DatabaseName=ldb;
torque.database.schema      = dbo
torque.database.user        = ldb_user
torque.database.password    = ldb_password

※schemaは、「dbo」と指定しなければならないことにに注意して下さい。
    →どこのDatabaseに接続するかはURLのParameterにて指定します。

※mssqlserver.jarの時とDriverのClassが変わっています。

  x  com.microsoft.jdbc.sqlserver.SQLServerDriver

  o  com.microsoft.sqlserver.jdbc.SQLServerDriver
