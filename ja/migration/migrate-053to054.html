【VersionUp-TIPS】
VersionUp時の運用についてのTIPSをここで紹介しておきます。
(DBFluteのVersionに関係なく以下の方法でOKです)


DBFluteの奨める「Project固有DBFlute」の構成を採用している場合：
  → How-to-Setupを参考


  xxxproject
      |
      +- .settings
      |
      +- dbflute_[dbname]          // DBFluteのProjectDirectory
      |     |
      |     |
      |     +- dfprop              // build-xxx.propertiesのDB接続情報や'map:{}'系Propertyを外だしできる
      |     +- log
      |     +- output
      |     +- playsql
      |     +- schema
      |     +- _project.bat ☆
      |     +- ...
      |
      +- mydbflute                 // DBFlute本体の格納場所
      |     |
      |     +- dbflute-0.4.4       // 古いVersionはいずれ削除してもよい。
      |     +- dbflute-0.4.5
      |     +- ...                 // _project.batには最新のDBFluteを設定しておく。
      | 
      +- src/main/java
      +- src/main/resources
      +- ...
      ...


1. DownloadしたDBFluteを解凍して、mydbflute以下に配置(dbflute-0.5.3)します。
2. _project.bat (or _project.sh) の DBFLUTE_HOME の指すVersionを 0.5.3 にする。
3. jdbc.bat / generate.bat を実行する。
