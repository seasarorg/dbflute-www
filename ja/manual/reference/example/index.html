DBFlute Exampleとは？

DBFluteの環境周りおよび実装周りの理解を一層深めてもらうための、実装にDBFluteを利用したEclipseプロジェクトです。Subversion からチェックアウトすることが可能で、実際にDBFluteのタスクの実行や、Example実装(JUnitのテストケース)を実行することができます。

(ほとんどの)Exampleは、スキーマ構造として ExampleDB を利用しています。

DIコンテナ

サポートされるDIコンテナとのExample。

H2 Database を組込みDBとして利用していて、依存ライブラリも組み込まれてるため(Mavenで管理していない)、SVNチェックアウトのみ ですぐにテスト実行(JUnit)できます。

dbflute-basic-example
  • Seasar(S2Container) を利用したExample
  • 日本語環境の基本プロジェクト
  • DB非依存処理の内部テストあり
DIコンテナごとの取扱い - Seasar(S2Container)
dbflute-spring-example
  • Spring Framework を利用したExample
  • 英語環境の基本プロジェクト
  • H2 Database のExampleとしての役割も兼ねる
DIコンテナごとの取扱い - Spring Framework
DBMSごとの取扱い - H2 Database
dbflute-guice-example
  • Google Guice を利用したExample
  • H2のシーケンスキャッシュのテストあり
  • テーブル区分値の自動デプロイのExample
  • shortCharHandlingのExample
DIコンテナごとの取扱い - Google Guice
dbflute-lucy-example
  • Lucy (T2プロジェクト)を利用したExample
  • JDK-6.0必須
DIコンテナごとの取扱い - T2 Lucy

データベース(DBMS)

サポートされるデータベース(DBMS)とのExample。

実行するにはそれぞれのデータベース環境の準備が必要です(readme.txt参照)。ストアドプロシージャやDB依存機能、DBMSごとの挙動の違いなどが参考になります。

dbflute-mysql-example
  • MySQL を利用したExample
  • 依存ライブラリは "Maven" (pom.xml)で管理されている
  • DB依存処理の内部テストあり
  • additionalSchema の Example (DB環境構築では、追加スキーマに対応するReplaceSchemaも実行すること)
  • 区分値機能のclassificationResourceのExample
  • ApplicationBehavior のライブラリプロジェクトとしての役割兼ねる
