検索ショートカットとは？

クラスの検索やHTMLファイルの検索など、IDE上で取り扱っているリソースを検索するショートカットです。 単なる名前による検索から階層や呼び出し元などプログラミングのエッセンスを使った検索ができます。

プログラム周り

クラスの検索
ctrl + shift + T (ctrl -> command in Mac)
キャメルケースコード補完と同じ要領で Eclipse 上で参照できるクラスを探し、そのファイルを開くことができる。
アスタリスク "*" で前方一致や後方一致、部分一致なども利用できる。 JARファイルの中のクラスは、(jadがあれば)逆コンパイルしたソースが表示される。
呼び出し元の検索
ctrl + shift + G (ctrl -> command in Mac)
選択したクラスやメソッドの呼び出し元を探すことができる。
階層構造の検索
ctrl + T (ctrl -> command in Mac)
選択したクラスやメソッドの階層を探すことができる。 主に、選択したクラスのサブクラスを探すのに便利。

リソース周り

リソース(ファイル)の検索
ctrl + shift + R
キャメルケースコード補完と同じ要領で Eclipse 上に存在しているファイルを探し、そのファイルを開くことができる。 (Mac では ctrl は command)
クラスファイルも検索できるので、Eclipse上で java として管理しているクラスであれば、クラスの検索(ctrl + shift + T)の代わりにもなる。(検索ノイズが減る)
