JDBCの設定について記述します。 


Apache Derby (derby)

'derby.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties(./dfprop/databaseInfoMap.dfprop) - for JDBC task}
map:{
    ; driver          = org.apache.derby.jdbc.EmbeddedDriver
    ; url             = jdbc:derby:../src/main/webapp/WEB-INF/data-derby;create=true
    ; schema          = LDB
    ; user            = ldb_user
    ; password        = ldb_password
}

※schemaは、大文字で指定しなければならないことにに注意して下さい。

※上記は「実行時にServiceを自動起動するモードの場合」の指定です。

    ; password        = ldb_password
}



PostgreSQL (postgresql)

'postgresql-8.1-407.jdbc3.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties(./dfprop/databaseInfoMap.dfprop) - for JDBC task}
map:{
    ; driver          = org.postgresql.Driver
    ; url             = jdbc:postgresql://localhost:5432/ldb
    ; schema          = 
    ; user            = ldb_user
    ; password        = ldb_password
}

※schemaは、指定する必要はありません。
    →どこのDatabaseに接続するかはURLにて指定します。

SQLServer (mssql)

'sqljdbc.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties(./dfprop/databaseInfoMap.dfprop) - for JDBC task}
map:{
    ; driver          = com.microsoft.sqlserver.jdbc.SQLServerDriver
    ; url             = jdbc:sqlserver://localhost:1433;DatabaseName=ldb;
    ; schema          = dbo
    ; user            = ldb_user
    ; password        = ldb_password
}

※schemaは、「dbo」と指定しなければならないことにに注意して下さい。
    →どこのDatabaseに接続するかはURLのParameterにて指定します。

※mssqlserver.jarの時とDriverのClassが変わっています。

  x  com.microsoft.jdbc.sqlserver.SQLServerDriver

  o  com.microsoft.sqlserver.jdbc.SQLServerDriver



Sybase (sybase)

'jconn3.jar'を利用します。

ex) database:ldb, user:ldb_user, password:ldb_password の場合：

{build.properties(./dfprop/databaseInfoMap.dfprop) - for JDBC task}
map:{
    ; driver          = com.sybase.jdbc3.jdbc.SybDriver
    ; url             = jdbc:sybase:Tds:localhost:4100
    ; schema          = dbo
    ; user            = ldb_user
    ; password        = ldb_password
}

※ldb_userのDefault-Databaseがldbであること。
