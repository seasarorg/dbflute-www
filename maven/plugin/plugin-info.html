Plugin documentation

Goals available for this plugin:

GoalDescription
dbflute:manageManageCommandPlugin provides manage goal to run manage.[sh|bat].
dbflute:create-clientCreateClientPlugin provides create-client goal to create dbflute client.
dbflute:jdbcJdbcCommandPlugin provides jdbc goal to run jdbc.[sh|bat].
dbflute:outside-sql-testOutsideSqlTestCommandPlugin provides outside-sql-test goal to run outside-sql-test.[sh|bat].
dbflute:generateGenerateCommandPlugin provides generate goal to run generate.[sh|bat].
dbflute:sql2entitySql2entityCommandPlugin provides sql2entity goal to run sql2entity.[sh|bat].
dbflute:downloadDownload Plugin provides download goal to download a zip file of dbflute.
dbflute:generate-crud-tablemetaGenerateCrudPlugin provides generate-crud goal to generate CRUD pages.
dbflute:docDocCommandPlugin provides doc goal to run doc.[sh|bat].
dbflute:generate-crudGenerateCrudPlugin provides generate-crud goal to generate CRUD pages.
dbflute:upgradeUpgradePlugin provides upgrade goal to download a zip file of dbflute and replace _project.*.
dbflute:replace-schemaReplaceSchemaCommandPlugin provides replace-schema goal to run replace-schema.[sh|bat].

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven2.0
JDK1.5
MemoryNo minimum requirement.
Disk SpaceNo minimum requirement.

Usage

You could run 'mvn -up' to get the latest version of this plugin, or specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.seasar.dbflute<groupId>
          <artifactId>dbflute-maven-plugin<artifactId>
          <version>1.0.0<version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.seasar.dbflute<groupId>
        <artifactId>dbflute-maven-plugin<artifactId>
        <version>1.0.0<version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
