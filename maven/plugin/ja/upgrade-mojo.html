dbflute:upgrade

Full name:

org.seasar.dbflute:dbflute-maven-plugin:1.0.0:upgrade

Description:

UpgradePlugin provides upgrade goal to download a zip file of dbflute and replace _project.*.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Optional Parameters

Name Type Since Description
clientProject String - (no description)
dbfluteClientDir File - (no description)
dbfluteVersion String - (no description)
downloadDirUrl String - (no description) Default value is: http://dbflute.seasar.org/download/dbflute/.
downloadFileExtension String - (no description) Default value is: .zip.
downloadFilePrefix String - (no description) Default value is: dbflute-.
mydbfluteDir File - (no description) Default value is: ${basedir}/mydbflute.
