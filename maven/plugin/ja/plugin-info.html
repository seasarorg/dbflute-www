Plugin documentation

Goals available for this plugin:

Goal Description
dbflute:create-client CreateClientPlugin provides create-client goal to create dbflute client.
dbflute:sql2entity Sql2entityCommandPlugin provides sql2entity goal to run sql2entity.[sh|bat].
dbflute:outside-sql-test OutsideSqlTestCommandPlugin provides outside-sql-test goal to run outside-sql-test.[sh|bat].
dbflute:jdbc JdbcCommandPlugin provides jdbc goal to run jdbc.[sh|bat].
dbflute:replace-schema ReplaceSchemaCommandPlugin provides replace-schema goal to run replace-schema.[sh|bat].
dbflute:upgrade UpgradePlugin provides upgrade goal to download a zip file of dbflute and replace _project.*.
dbflute:generate GenerateCommandPlugin provides generate goal to run generate.[sh|bat].
dbflute:generate-crud GenerateCrudPlugin provides generate-crud goal to generate CRUD pages.
dbflute:download Download Plugin provides download goal to download a zip file of dbflute.
dbflute:doc DocCommandPlugin provides doc goal to run doc.[sh|bat].
dbflute:generate-crud-tablemeta GenerateCrudPlugin provides generate-crud goal to generate CRUD pages.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 2.0
JDK 1.5
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You could run 'mvn -up' to get the latest version of this plugin, or specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.seasar.dbflute<groupId>
          <artifactId>maven-dbflute-plugin<artifactId>
          <version>0.4.0-SNAPSHOT<version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.seasar.dbflute<groupId>
        <artifactId>maven-dbflute-plugin<artifactId>
        <version>0.4.0-SNAPSHOT<version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
For more information, see "Guide to Configuring Plug-ins"
