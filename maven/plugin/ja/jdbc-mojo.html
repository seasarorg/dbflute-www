dbflute:jdbc

Full name:

org.seasar.dbflute:maven-dbflute-plugin:0.4.0-SNAPSHOT:jdbc

Description:

JdbcCommandPlugin provides jdbc goal to run jdbc.[sh|bat].

Attributes:

  • Requires a Maven 2.0 project to be executed.
