DBFlute現場指向O/Rマッパ

dbflute:replace-schema

概要

ReplaceSchemaCommandPlugin provides replace-schema goal to run replace-schema.[sh|bat].

条件

  • Requires a Maven 2.0 project to be executed.

パラメータ
