DBMetaの基本

DBMetaとは？

DBMetaは、Entityが表現する構造(テーブルや外だしSQL(OutsideSql))のメタ情報を表現するオブジェクトです。

Singletonオブジェクトであり、Entityから取得してもDBMetaのgetInstance()メソッドで取得しても同じインスタンスになります。 また、全てのDBMetaは「DBMetaインターフェース」を実装しているので抽象化して利用することが可能です。 

ex) 会員(Member)のDBMetaを取得// Get from singleton instance method
MemberDbm dbmeta = MemberDbm.getInstance();

// Get from entity as DBMeta interface
DBMeta dbmeta = member.getDBMeta();

DBMetaの種類

DomainEntityのDBMeta
テーブルに対応するDBMeta。Generateタスクで自動生成される。
CustomizeEntityのDBMeta
外だしSQL(OutsideSql)に対応するDBMetad。Sql2Entityタスクで自動生成される。
